CSS Selectors and Common CSS Properties

Table 1: CSS Selectors

Selector Type Example Text Description
Type selector p Targets every element of one HTML type.
Class .highlight Selects elements with the specified class
ID selector #id Selects an element with the specified ID
Grouping Selector h1, h2, p Applies the same style to multiple selectors.
Universal Selector * Selects all elements.

Table 2: Common CSS Properties

Property Example Text Description
Color color: red; Sets the text color of an element.
Font-size font-size: 16px; Sets the font size of an element.
Font-family font-family: Arial, sans-serif; Sets the font family of an element.
Background-color background-color: #BA807BFF; Sets the background color of an element.
Text-align text-align: center; Sets the text alignment of an element.
Font-weight font-weight: bold; Sets the font weight of an element.
Margin margin: 20px; Sets the margin of an element.
Padding padding: 10px; Sets the padding of an element.
Line Height line-height: 1.5; Sets the line height of an element.
Border border: 1px solid #ccc; Sets the border of an element.
